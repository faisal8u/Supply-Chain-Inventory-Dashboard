# Supply Chain Inventory & Performance Dashboard

## Problem Statement
A computer trading company holds AED 25 million in aging warehouse stock.
This dashboard was built to help management identify, monitor, and reduce
old stock through data-driven decision making.

## Dashboard KPIs
- Total Old Stock Value (AED)
- Dead Stock Value — items with no movement for 1+ year
- Slow Moving Stock % — items with no movement in 90+ days
- Inventory Turnover Ratio
- Supplier On-Time Delivery Rate %

## Visuals
- Stock Value by Product Category (Bar Chart)
- Stock Age Breakdown (Donut Chart)
- Old Stock Value Trend Over Time (Line Chart)
- Top Slow Moving Items Table

## Tools Used
- Power BI Desktop
- Power Query (data cleaning)
- DAX (KPI calculations)
- Excel (data source)

## Dataset
Dummy dataset simulating real warehouse inventory, sales transactions,
supplier performance, and monthly stock snapshots across 120 products
